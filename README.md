## Sobre los Scripts

A menos que se indique lo contrario, los scripts que resuelven algunos de los ejercicios
de las tareas están escritos en Python. De manera práctica, todo controlado desde un mismo
entorno virtual desde la carpeta raiz.

Se sugiere de sobremanera usar [uv](https://docs.astral.sh/uv/) para ejecutarlos e instalar las bibliotecas de la manera
más rápida y sencilla posible. Si es el caso, se hace lo siguiente:

```bash
uv sync
source .venv/bin/activate
```

Posteriormente, se siguen los comandos que se encuentren en README de cada tarea.
